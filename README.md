# Salary Prediction using Linear Regression

## Project Overview

This project predicts employee salary based on years of experience using Linear Regression.

## Tech Stack

* Python
* Pandas
* NumPy
* Scikit-learn
* Streamlit
* Joblib

## Features

* Salary prediction using ML
* End-to-end model training
* Model saving using Joblib
* Streamlit web app for prediction

## Run Project

pip install -r requirements.txt

streamlit run app/app.py
